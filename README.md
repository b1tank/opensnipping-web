<h1 align="center">OpenSnipping 小画剪</h1>

<p align="center">
  A modern, browser-based screenshot and annotation tool.<br>
  一款现代化的浏览器截图和标注工具。<br>
  Built with pure HTML/CSS/JavaScript — no frameworks, no build step.
</p>

---

## ✨ Features

- Screen capture via browser's screen-sharing picker
- Crop to a selected region
- Annotate with pen + eraser
- Undo/redo
- Copy to clipboard (PNG) and save to file

## 🔒 Privacy

OpenSnipping runs entirely in your browser. Captured images are processed locally and never uploaded.

## 📝 Notes

- Screen capture requires user permission via the browser UI
- Clipboard image copy requires a compatible browser/context (some browsers restrict Clipboard APIs)
